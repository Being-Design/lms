== Description ==

Manages license and update process for BuddyBoss products.

== Installation ==

1. Visit 'Plugins > Add New'
2. Click 'Upload Plugin'
3. Upload the file 'buddyboss-updater.zip'
4. Activate BuddyBoss Updater from your Plugins page.
